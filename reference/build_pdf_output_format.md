# Build an RMarkdown output format for PDF files.

Build an RMarkdown output format for PDF files.

## Usage

``` r
build_pdf_output_format(header, output_options = NULL)
```

## Arguments

- header:

  Metadata from the RMarkdown file.

- output_options:

  A list of extra output options for `\link{pdf_document}`.

## Value

An RMarkdown output format object.
