# Initialize semester schedule

Convert a semester data object into a schedule suitable for generating
assignments.

## Usage

``` r
init_schedule(semester)
```

## Arguments

- semester:

  A semester object (a list) returned from
  [`load_semester_db()`](https://jonathan-g.github.io/semestr/reference/load_semester_db.md)

## Value

A data frame containing the semester schedule.
